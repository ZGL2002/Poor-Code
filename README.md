# PoorCode

> 一个终端 AI 编程助手——自己动手写的"平替版" Claude Code。

## 简介

PoorCode 是用 Python 实现的命令行 AI 编程助手。它能理解自然语言指令，在终端中直接帮你读代码、改代码、执行命令，像一个真正的人类结对编程伙伴一样工作。

## 特性

- 🧠 **自然语言交互**——用中文聊天，让 AI 理解你的编程需求
- 🔧 **工具调用**——读取文件、编辑代码、搜索内容、执行 Shell 命令
- 💻 **终端原生**——在 tmux 中运行，端到端真实体验
- 🐍 **纯 Python 实现**——轻量、可定制、易于理解

## 快速开始

```bash
# 克隆项目
git clone <repo-url>
cd poorcode

# 安装依赖
pip install -r requirements.txt

# 启动
python poorcode.py
```

## 使用方式

在终端中直接输入你的需求，例如：

```
> 帮我看看当前目录下有哪些 Python 文件
> 把这个函数的参数改成关键字参数
> 运行测试并告诉我结果
```
